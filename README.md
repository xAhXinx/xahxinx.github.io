# timer.github.io
## 快捷键说明
Spacebar        - Start/Switch 计时器（对辩和自由辩是 Switch）
P               - Pause当前页面的计时器
R               - Reset当前页面的计时器
"Shift" + "+/=" - 在"教练指导环节"和"评审点评环节"添加时长1分钟
→               - Next 下一页
←               - Back 上一页
C               - 跳转到教练指导环节
Esc             - 从教练指导环节退出
